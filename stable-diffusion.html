Bons Prompts

Ecrire de bons prompts

La méthode pour composer des descriptions et prompts qui généreronts de belles images avec Stable Diffsuion

Introduction

Stable Diffusion est une technologie d'IA fascinante qui permet de créer des images à partir de descriptions textuelles. Que vous soyez un artiste numérique, un designer ou simplement un curieux, savoir comment formuler des prompts efficaces peut transformer vos créations. Dans ce guide, nous explorerons les meilleures pratiques pour rédiger des prompts qui génèrent des images impressionnantes et fidèles à vos attentes.

1. Comprendre Stable Diffusion

Stable Diffusion est un modèle de génération d’images basé sur l’IA, entraîné avec des millions d'exemples de textes et d'images. En fournissant une description textuelle, ou « prompt », à l’IA, vous lui donnez une directive sur ce que vous souhaitez visualiser. Plus votre prompt est précis et détaillé, plus l’image générée sera proche de ce que vous imaginez.

2. Langue du Prompt : Anglais vs Français

Bien que Stable Diffusion soit capable de comprendre le français, il a été principalement entraîné avec des descriptions en anglais. Pour des résultats optimaux, il est donc conseillé d’écrire vos prompts en anglais. Vous pouvez utiliser des outils de traduction comme Deepl ou chatGPT pour convertir vos descriptions du français à l'anglais.

3. La Structure d’un Bon Prompt

Pour obtenir des images de haute qualité et spécifiquement adaptées à vos besoins, il est crucial de structurer votre prompt correctement. Voici les éléments clés à inclure :

  • Sujet (obligatoire) : Décrivez clairement le sujet principal de l’image. Plus vous êtes précis, mieux c’est. Par exemple : « Un garçon geek programmant devant un ordinateur ».
  • Média : Indiquez le type de représentation visuelle souhaité. Cela peut inclure des styles comme la peinture numérique, la photographie, ou la peinture à l'huile. Exemple : « Peinture numérique ».
  • Style : Définissez le style ou l’ambiance de l’image. Cela peut être un style artistique spécifique ou une atmosphère générale. Exemple : « Anime japonais ».
  • Artiste : Mentionnez un artiste dont le style vous inspire, si cela peut influencer le rendu. Exemple : « Par Makoto Shinkai et le Studio Ghibli ».
  • Site Web : Ajouter des sites connus pour leur qualité d’illustration peut aider à orienter le style. Exemple : « Artstation ».
  • Résolution et Qualité : Indiquez la résolution souhaitée pour l’image. Des mots-clés comme « 4k » ou « 8k » peuvent améliorer la qualité. Exemple : « 8k ».
  • Détails Supplémentaires : Ajoutez des détails pour peaufiner l’image, comme l’éclairage ou la précision. Exemple : « Hautement détaillé, qualité incroyable ».
  • Couleurs : Précisez les tonalités ou la palette de couleurs désirées. Exemple : « Couleurs vives ».

4. Exemples de Prompts

Pour mieux comprendre comment ces éléments s’intègrent, voici quelques exemples :

  • Prompt détaillé : « Peinture numérique d’un garçon geek qui programme devant un ordinateur, anime japonais, par Makoto Shinkai et le Studio Ghibli, Artstation, 8k, hautement détaillé, qualité incroyable, couleurs vives ».
  • Prompt avec variations : « Rendu 3D d’un garçon geek qui programme devant un ordinateur, American comics, par Todd McFarlane et le Studio Marvel, DeviantArt, 4k, hautement détaillé, éclairage cinématique, ambiance sombre ».

Ces exemples montrent comment l’ajout de différents mots-clés peut modifier l’apparence et le style de l’image générée.

5. Liste de Mots-Clés à Utiliser

Voici quelques mots-clés utiles pour divers aspects de vos prompts :

  • Média : Digital painting, Realistic Photography, Oil Painting.
  • Artistes : John Collier, Greg Rutkowski, Ghibli.
  • Sites Web : Artstation, Deviant Art.
  • Résolution : 8k, Unreal Engine.
  • Autres Détails : Dramatic lighting, Vivid colors.

Conclusion

Créer des prompts efficaces pour Stable Diffusion est une combinaison d’art et de technique. En structurant vos descriptions avec soin et en utilisant les bons mots-clés, vous pouvez obtenir des images qui répondent exactement à vos attentes. N’oubliez pas que Stable Diffusion est flexible et expérimenter avec différentes combinaisons peut mener à des découvertes surprenantes. Amusez-vous et laissez libre cours à votre créativité pour générer des visuels captivants et uniques !

Prompt Image

Trouver le prompt d'une image

Comment obtenir le prompt d’une image avec Stable Diffusion

Introduction

Vous avez découvert une image générée par IA qui vous fascine et vous vous demandez quel était le prompt utilisé pour la créer ? Cet article vous guide pour retrouver ou deviner le prompt d'une image générée avec Stable Diffusion. Apprenez à utiliser les métadonnées et des outils d'IA pour découvrir le secret derrière vos images préférées.

1. Trouver le Prompt dans les Métadonnées

Les métadonnées d’une image peuvent contenir le prompt original et les paramètres de génération. Voici comment les obtenir :

  • Utiliser PNG Info dans Automatic1111 : Téléchargez l’image dans l’onglet PNG Info pour voir les métadonnées. Si disponibles, le prompt et les paramètres apparaîtront.
  • PNG Info en ligne : Utilisez le site [Linaqruf PNG Info] (https://huggingface.co/spaces/Linaqruf/pnginfo) pour extraire les métadonnées d’images générées avec différentes interfaces.
  • Utiliser Jimpl : Un autre outil gratuit pour extraire les métadonnées est [Jimpl] (https://jimpl.com/) Téléchargez l’image et voyez les informations disponibles.

Note : Ces méthodes ne fonctionnent que pour les images générées avec Stable Diffusion et ses interfaces compatibles. Elles ne s'appliquent pas aux images créées avec d’autres IA comme Midjourney ou DALL·E.

2. Deviner le Prompt

Quand les métadonnées ne sont pas disponibles, vous pouvez deviner le prompt en utilisant des outils de description d’image :

  • CLIP Interrogator : Utilisez un outil comme CLIP Interrogator pour convertir une image en description textuelle. Vous pouvez utiliser [Automatic1111] (https://github.com/pharmapsychotic/clip-interrogator-ext) ou [Hugging Face] (https://replicate.com/pharmapsychotic/clip-interrogator) .
  • Fonction Describe : Des outils comme Fooocus et Midjourney offrent des fonctions pour générer des descriptions de l’image. Sur Midjourney, utilisez la commande /describe sur Discord.
  • Image to Prompt avec ChatGPT : Pour les abonnés ChatGPT Plus, vous pouvez demander à ChatGPT de décrire une image et générer un prompt basé sur cette description.

3. Conseils pour Améliorer les Résultats

Les outils de génération de prompts ne sont pas toujours parfaits. Voici quelques conseils pour affiner les résultats :

  • Modifiez le Prompt : N’hésitez pas à ajuster les prompts générés pour mieux correspondre à l’image.
  • Choisissez le Modèle Approprié : Utilisez un modèle Stable Diffusion adapté au type d’image que vous souhaitez obtenir (anime, photographie, illustration, etc.).
  • Utilisez l'Image comme Prompt : Vous pouvez aussi utiliser une image comme prompt pour reprendre son style ou ses couleurs.

Conclusion

Découvrir le prompt d’une image générée par IA peut être fascinant et utile pour vos propres créations. En utilisant les métadonnées lorsque disponibles ou en devinant le prompt avec des outils d’IA, vous pouvez mieux comprendre et reproduire les images qui vous inspirent. Explorez ces méthodes et laissez libre cours à votre créativité !

Prompts Négatifs

Maîtriser les Prompts Négatifs

Le guide du Negative Prompting pour améliorez la qualité de vos Images avec Stable Diffusion

Introduction

Vous pouvez améliorer la qualité des images générées avec Stable Diffusion grâce à l'utilisation de "Negative Prompts" (prompts négatifs). Ce guide vous aidera à comprendre et à maîtriser cette technique pour contrôler ce que vous ne voulez pas voir dans vos créations.

1. Qu'est-ce qu'un Prompt Négatif ?

Un prompt négatif est une instruction donnée à l'IA pour indiquer ce que vous ne voulez pas voir dans l'image générée. Contrairement au prompt principal, qui guide le processus de génération, le prompt négatif aide à éliminer certains éléments ou à influencer le style de l'image.

2. Comment Utiliser les Prompts Négatifs

Les prompts négatifs permettent de :

  • Éliminer des éléments indésirables : Par exemple, supprimer des voitures d'une image de rue en utilisant le prompt négatif "car".
  • Modifier le style : Pour un style plus photoréaliste, utilisez des prompts négatifs comme "painting" ou "cartoon".
  • Faire des ajustements subtils : Utilisez "wind" pour corriger des cheveux flottants à cause du vent.

3. Exemples Pratiques

Voici des exemples concrets :

  • Supprimer des éléments : En utilisant des prompts négatifs tels que "Flower" pour enlever des fleurs d'une image.
  • Changer de style : Pour passer d'une image en CG à une image photoréaliste, ajoutez des prompts négatifs comme "cg" ou "computer graphic".

4. Stable Diffusion v2 et le Negative Prompting

Avec l'arrivée de Stable Diffusion v2, les prompts négatifs sont devenus essentiels pour améliorer la qualité des images. L'ajout de quelques prompts négatifs peut considérablement affiner les résultats.

5. Les Meilleurs Prompts Négatifs

Voici quelques prompts négatifs universels pour améliorer la qualité :

blur, ugly, tiling, bad art, poorly drawn hands, poorly drawn feet, poorly drawn face, out of frame, extra limbs, disfigured, body out of frame, bad anatomy, watermark, signature, cut off, low contrast, underexposed, overexposed, beginner, amateur

6. Différents Usages

Pour différents usages, adaptez les prompts négatifs selon le type d'image :

  • Photorealistic : unnatural, unrealistic, cartoon, illustration
  • Paintings and Illustrations : bad art, photorealistic, realism
  • Landscapes : plain background, boring, low quality
  • Urban Scenes : oversaturated, low contrast, creepy
  • Portraits : disfigured, bad anatomy, cross-eyed

Conclusion

Les prompts négatifs sont un outil puissant pour affiner vos créations avec Stable Diffusion. Ils vous permettent de contrôler ce que vous ne voulez pas voir dans vos images et d'influencer le style de manière précise. Continuez à expérimenter et à découvrir comment optimiser vos images grâce à cette technique.

Prompt Weight

Prompt Weight : le poids des mots

Utilisez le Prompt weighting pour améliorer vos prompts

Introduction

Stable Diffusion a transformé la création d'images à partir de descriptions textuelles. Pour améliorer encore vos résultats, découvrez le Prompt Weighting, une technique simple mais puissante pour contrôler la génération d'images en attribuant différents poids aux mots de votre prompt.

1. Qu'est-ce que le Prompt Weighting ?

Le Prompt Weighting permet d'ajuster l'importance des mots dans votre prompt. Cela vous aide à guider l'IA pour qu'elle se concentre davantage sur certains éléments de l'image générée.

2. Comment ça marche ?

Pour modifier le poids d'un mot, vous devez utiliser une syntaxe spécifique dans votre prompt :

  • (mot:poids) : Augmente le poids du mot
  • [mot] : Diminue le poids du mot

Par exemple, dans le prompt "Photography of a woman with (blue) eyes", le mot "blue" est mis en avant, tandis que "Photography of a woman with [blue] eyes" diminue son importance.

3. Exemples Pratiques

Considérons un prompt simple : "Woman, Beach, Pizza". Sans ajustement de poids, chaque mot a une importance égale. Voici comment les images changent en modifiant le poids :

  • Sans poids spécifique
  • En augmentant le poids de "Woman"
  • En augmentant le poids de "Beach"
  • En augmentant le poids de "Pizza"

Augmenter le poids de "Beach" montre plus de plage, tandis qu'augmenter le poids de "Pizza" rend la pizza plus grande.

4. Comment l'utiliser dans différentes interfaces ?

Voici comment utiliser le Prompt Weighting dans diverses interfaces :

Automatic1111

Utilisez les parenthèses pour ajuster les poids : (mot:1.5) pour augmenter et [mot] pour diminuer.

Fooocus

Similar à Automatic1111, avec des raccourcis pour ajuster les poids.

ComfyUI

Utilise une syntaxe similaire : (mot:1.1). Les poids doivent souvent être ajustés plus finement que dans Automatic1111.

Conclusion

Le Prompt Weighting est un outil puissant pour affiner vos créations avec Stable Diffusion. En ajustant le poids des mots, vous pouvez influencer considérablement le résultat final. Expérimentez avec différents poids pour explorer tout le potentiel créatif de cette fonctionnalité.

Portraits réalistes

Comment crées des photos réalistes avec SDXL

Introduction

Stable Diffusion XL (SDXL) permet de générer des portraits photoréalistes de haute qualité. Ce guide vous aidera à optimiser vos prompts pour obtenir des photos aussi réalistes que possibles.

1. Optimiser vos Prompts

Pour créer des portraits réalistes avec SDXL, commencez par un prompt simple et clair. Par exemple :

Portrait of a blond woman wearing a white shirt

Ajoutez le mot photographic pour spécifier que vous souhaitez un rendu photographique :

Photographic portrait of a blond woman wearing a white shirt

Evitez d'utiliser le mot photorealistic car il est souvent associé à des images générées et non à des photos réelles.

2. Utiliser les Prompts Négatifs

Pour éviter les styles non photographiques, ajoutez des mots-clés négatifs à votre prompt :

illustration, cartoon, anime, 3d render, painting, crayon, sketch, graphite, impressionist, unreal engine

3. Mots Clés pour Photos

Affinez vos résultats en utilisant des mots clés photographiques :

  • Éclairage : natural lighting, rim lighting, Rembrandt lighting
  • Appareils photo : shot with GoPro, Polaroid SX-70, Nikon D750
  • Types de film : Kodachrome, Fujifilm Superia, Film grain
  • Distance focale : 15mm, 80mm, 200mm
  • Cadrages : head and shoulders portrait, Extreme close up, Upper body

Ajoutez également des détails comme l’apparence de la peau :

  • skin pores, skin blemished, freckles, wrinkles

Spécifiez la direction du regard :

  • Looking at viewer, Looking away, Looking up

Incorporez des éléments et des props pour plus de réalisme :

  • holding a coffee mug, wearing ancient jewelry, under a crystal chandelier

4. Exemples de Prompts Complets

Combinez les éléments pour créer des prompts détaillés :

  • Photo of an african-american man in New York, head and shoulders portrait, shot with Fujifilm Superia, candid street photography
  • Close up portrait of a female barista, detailed photograph shot on Kodak, amazing skin details
  • Portrait of a Japanese yoga teacher, Instagram photography, traditional Japanese style room, eyes closed

5. Modèles Photoréalistes

Pour améliorer la qualité photoréaliste, utilisez des modèles spécifiques :

  • RealVisXL : Combine plusieurs modèles pour des visages naturels.
  • Realistic Stock Photo : Efficace pour les portraits serrés.
  • Copax TimeLessXL : Génère des images réalistes avec une variété de styles.
  • Realities Edge XL : Améliore les détails et la texture pour un photoréalisme optimal.

Conclusion

En utilisant les techniques et modèles décrits, vous pouvez créer des portraits photoréalistes impressionnants avec SDXL. Expérimentez avec les différents mots clés et modèles pour obtenir des résultats qui répondent à vos attentes créatives.

Prompts stylés

Des prompts qui ont du style

Une bibliothèque de Styles pour enrichir vos prompts SDXL

Stable Diffusion XL (SDXL) a introduit une manière simple d'améliorer la qualité et le style de vos images grâce à des styles prédéfinis. Que vous soyez débutant ou utilisateur avancé, ces styles vous permettent de créer des images visuellement captivantes en ajoutant simplement un style à votre prompt de base. Dans cet article, nous vous expliquerons comment utiliser ces styles et vous donnerons des exemples pratiques pour commencer.

Pourquoi Utiliser des Styles ?

Les styles sont des ensembles de paramètres qui définissent un type de rendu visuel spécifique. Au lieu de passer du temps à rédiger des prompts longs et complexes, les styles vous permettent d'obtenir des résultats rapides et élégants en quelques clics. Cela vous offre une grande flexibilité, en particulier si vous êtes novice en génération d'images IA.

Comment Utiliser les Styles ?

Il existe plusieurs manières d'utiliser les styles dans Stable Diffusion, selon l'interface ou l'outil que vous utilisez. Voici quelques exemples :

Utilisation des Styles dans Fooocus

Fooocus intègre nativement des collections de styles comme les Diva Styles et les MK-Styles. Pour les utiliser :

  • Cliquez sur l'option Advanced sous le champ du prompt.
  • Accédez à l'onglet Styles pour choisir un ou plusieurs styles à appliquer à votre image.

Exemple de Style : Diva

Prompt : a cat with a hat (un chat avec un chapeau) + Style Diva

Utilisation des Styles dans Automatic1111

Avec l'extension Style Selector XL pour Automatic1111, vous pouvez facilement accéder à une liste de styles prêts à l'emploi :

  • Activez l'extension en cochant la case Enable Style Selector dans les options de génération img2img.
  • Choisissez un style parmi la liste disponible.

Voir le tutoriel d'installation de l'extension

Utilisation des Styles dans ComfyUI

Dans ComfyUI, l'extension SDXL Prompt Styler vous permet d'appliquer des styles à vos prompts via un node dédié dans votre workflow.

Collections de Styles Disponibles

Voici quelques collections populaires de styles que vous pouvez utiliser dans vos générations :

  • Stability AI Styles : Ce sont les 17 styles originaux proposés lors du lancement de SDXL. Disponibles sur DreamStudio et compatibles avec plusieurs interfaces.
  • Diva Styles : Une collection visuellement percutante qui ajoute une touche de sophistication à vos images.
  • MK-Styles : Connu pour des rendus dynamiques et colorés, parfait pour des créations vibrantes.

Diffusion rapide Stable XL sur TPU v5e ⚡

Vous souhaitez réaliser des inférences rapides et efficaces avec Stable Diffusion XL ? Grâce à Google Cloud TPU v5e, vous pouvez générer des images haute qualité (1024×1024) de manière rapide et rentable. Cette technologie est disponible via une démo pratique sur Hugging Face.
